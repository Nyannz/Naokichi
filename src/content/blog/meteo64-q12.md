## 問題のテーマ

気象業務法第17条・第18条に基づく予報業務許可の要件に関する正誤問題。

## 気象業務法第18条（許可の基準）

許可要件として条文に明記されているのは**2点**：

1. **予報資料の収集・解析の施設と要員**を有すること
   - この中に気象予報士の設置義務も含まれる
2. **気象庁の警報事項を迅速に受信する施設と要員**を有すること

## 各選択肢の解説

### (a) ○（正しい）

気象庁の警報事項を迅速に受信できる施設と要員 → **第18条の要件** ✓

### (b) ○（正しい）←ここを誤りと判断しがち

予報業務に必要な**予報資料の収集のための施設と要員** → **第18条の要件** ✓

「資料収集は要件に含まれない」と思いがちだが、条文に明記されている。

### (c) ×（誤り）

予報の「解説」のための施設と要員 → **要件ではない**  
解説は予報業務そのものではなく、法的な許可要件に含まれない。

### (d) ×（誤り）←ここが最大のひっかけ

予報事項を利用者に「伝達」するための施設 → **要件ではない**

**重要な区別：**

| 要件 | 具体例 |
|------|-------|
| 警報事項を**受信**する | 気象庁の警報を受け取る施設 |
| 予報資料を**収集**する | 数値予報資料などの入力 |
| 予報事項を**伝達**する | ❌ 要件ではない |

「入力側（受信・収集）は要件、出力側（伝達）は要件ではない」。

## 正解：②（a正・b正・c誤・d誤）

## 覚え方

```
許可要件 = 入ってくるものだけ
  ├─ 警報事項が「入ってくる」施設と要員 ✓
  └─ 予報資料が「入ってくる」施設と要員 ✓

出ていくもの（利用者への伝達）は要件ではない ✗
```

## ポイントまとめ

- 要件は条文上**2本柱**（警報受信 + 予報資料収集・解析）
- 「解説」「伝達」は許可要件に含まれない
- 気象予報士の設置は「予報資料の収集・解析の要員」の中に含まれる
